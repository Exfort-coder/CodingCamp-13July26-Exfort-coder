# Task

Create a **To-Do List Life Dashboard** as a standalone, client-side web app.

## Must-haves (MVP)

1. **Greeting**
   - Show current time and date
   - Show a greeting based on time of day
2. **Focus Timer**
   - 25-minute timer (Pomodoro)
   - Start, stop, reset
3. **To-Do List**
   - Add tasks
   - Edit tasks
   - Mark tasks as done
   - Delete tasks
   - Persist tasks using **LocalStorage**
4. **Quick Links**
   - Buttons/links that open favorite websites
   - Save links in **LocalStorage**

## Added features

- Light/Dark mode
- Custom name in greeting
- Change Pomodoro time
- Sort tasks

## Constraints

- HTML structure
- CSS styling
- Vanilla JavaScript only (no frameworks)
- LocalStorage only (no backend)
- Must work in modern browsers

## Deliverables

- `index.html`
- `css/style.css` (single CSS file)
- `js/app.js` (single JS file)
- Updated/created `README.md`
